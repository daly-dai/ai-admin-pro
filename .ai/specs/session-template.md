# 会话交接模板

> 每个 Task 完成后由 AI 生成，用于跨会话传递上下文，防止新会话丢失决策记录

## 使用说明

- **生成时机**：Task 执行完毕后（无论成功/失败）
- **生成者**：AI（用户可要求补充）
- **用途**：新会话启动时读取，快速恢复开发上下文
- **关联文件**：
  - `specs/[feature]/spec.md` — Task 定义和完成标准
  - `specs/[feature]/progress.md` — 整体进度追踪

## 状态符号

| 符号 | 含义        |
| ---- | ----------- |
| ✅   | 完成        |
| ⏳   | 部分完成    |
| 🔴   | 失败/未完成 |

## 填写原则

1. **精炼**：每项 1-2 句话，避免冗余
2. **具体**：文件路径、错误信息要完整
3. **可追溯**：决策原因必须记录，防止被"纠正"
4. **面向下一会话**：重点是"新 AI 需要知道什么"
