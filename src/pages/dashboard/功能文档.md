# 时点规模 & 国债收益率趋势监控大屏

> 版本：v2.0 | 更新日期：2026-05-26

---

## 1. 概述

4 模块大屏仪表盘，每个模块展示一条双 Y 轴折线图（左轴：时点规模，右轴：国债收益率）。支持 2×2 网格布局、全局筛选、分模块独立翻页、弹框放大查看、数据缓存去重。

**适用场景**：金融从业者日常监控时点规模、DV01 风险、修正久期、YTM 收益率的趋势变化。

**技术栈**：React 18 + TypeScript + ECharts 6 + Zustand + dayjs + antd 5

## 4. 功能详情

### 4.1 全局筛选栏

| 控件     | 组件         | 影响范围                    |
| -------- | ------------ | --------------------------- |
| 结束日期 | `DatePicker` | 全部 4 个模块               |
| 账户类型 | `Select`     | 全部 4 个模块的时点规模数据 |
| 债权类型 | `Select`     | 全部 4 个模块的时点规模数据 |
| 币种     | `Select`     | 全部 4 个模块的时点规模数据 |

**行为**：任一筛选项变更 → 立即清空全部 4 个模块的数据 → 各自以新条件重新请求第一批数据。缓存不清除（翻回旧区间仍可命中）。

### 4.2 翻页（上一页）

- 每个模块独立拥有「上一页」按钮，位于模块卡片标题栏
- 点击「上一页」→ 计算当前最早日期 - 1 天 → 以新 endDate 请求数据 → 新数据追加（unshift）到已有数据头部 → 图表时间范围自动扩展
- 4 个模块的翻页互不影响：时点规模累计 90 天，DV01 仍可以只有 30 天
- 按钮在以下情况禁用：
  - 当前模块正在加载中
  - 后端返回数据量 < 约定 pageSize（表示已到历史数据边界）

### 4.3 dataZoom 缩放

- **底部滑块（slider）**：鼠标拖拽两端把手调整显示范围
- **图表内手势（inside）**：鼠标滚轮缩放、拖拽平移
- **纯前端操作**，不触发任何接口请求
- **默认展示全部已加载数据**（start: 0, end: 100）

### 4.4 弹框放大查看

- 每个模块卡片标题栏右侧有「⛶」展开按钮
- 点击后弹出 90vw 宽度的 Modal
- 弹框内图表高度 500px（vs 卡片内 260px），dataZoom 操作更精细
- 弹框内同样有「上一页」按钮，可直接翻页
- 弹框关闭后卡片数据保持同步（共享同一 store，无需额外请求）

### 4.5 Y 轴动态范围

- 两个 Y 轴均不固定从 0 开始
- 根据当前数据的 min/max 动态计算显示范围，上下各留 10% 余量

| Y 轴       | 精度  | 示例                                           |
| ---------- | ----- | ---------------------------------------------- |
| 时点规模   | 整数  | 数据范围 3102 ~ 4850 → Y轴显示 3000 ~ 5000     |
| 国债收益率 | 0.01% | 数据范围 2.65% ~ 3.42% → Y轴显示 2.58% ~ 3.49% |

- 单值场景：以该值为中心 ±10% 扩展，避免图表无高度
- 空数据场景：ECharts 自行处理，不设置 min/max

### 4.6 数据缓存

缓存存储在 Zustand store 中（内存级，页面刷新后丢失）。

**时点规模缓存**：

```
key = `${endDate}_${pageSize}_${accountType}_${debtType}_${currency}_${moduleType}`
```

每个模块独立隔离，相同参数的请求只发起一次。

**国债收益率缓存**：

```
key = `${endDate}_${pageSize}`
```

不含 `moduleType`，4 个模块共享。第一个模块请求后写入缓存，其余 3 个模块直接命中，**避免 4 次重复请求**。

**缓存生命周期**：

- 筛选条件变更不清除缓存
- 翻回已看过的区间 → 命中缓存 → 不发起网络请求
- 页面刷新后缓存清空

## 5. 接口约定

### 5.1 时点规模接口

```
GET/POST /api/dashboard/scale
```

| 参数          | 类型     | 必填 | 说明                                  |
| ------------- | -------- | ---- | ------------------------------------- |
| `endDate`     | `string` | 是   | 截止日期 `YYYY-MM-DD`                 |
| `pageSize`    | `number` | 是   | 返回条数 30/60/90                     |
| `accountType` | `string` | 是   | 账户类型                              |
| `debtType`    | `string` | 是   | 债权类型                              |
| `currency`    | `string` | 是   | 币种                                  |
| `moduleType`  | `string` | 是   | `scale` / `dv01` / `duration` / `ytm` |

**响应**：

```json
[
  { "timePoint": "2026-04-27", "indicatorValue": 3245.5 },
  { "timePoint": "2026-04-28", "indicatorValue": 3310.2 }
]
```

- `timePoint`：日期 `YYYY-MM-DD`
- `indicatorValue`：规模数值
- 按 `timePoint` **升序**返回

### 5.2 国债收益率接口

```
GET/POST /api/dashboard/treasury-yield
```

| 参数       | 类型     | 必填 | 说明                  |
| ---------- | -------- | ---- | --------------------- |
| `endDate`  | `string` | 是   | 截止日期 `YYYY-MM-DD` |
| `pageSize` | `number` | 是   | 返回条数 30/60/90     |

**响应**：

```json
[
  { "dataDate": "2026-04-27", "yieldRate": 2.6531 },
  { "dataDate": "2026-04-28", "yieldRate": 2.671 }
]
```

- `dataDate`：日期 `YYYY-MM-DD`
- `yieldRate`：收益率（百分比数值，如 `2.65` 表示 2.65%）
- 按 `dataDate` **升序**返回

### 5.3 后端字典接口（pageSize 配置）

后端需提供字典接口返回当前 `pageSize` 配置值（30/60/90）。前端初始化时读取并设置。字典值变更时，下次进入页面生效，现有缓存因 key 中 `pageSize` 变化自动失效。

## 8. 边界情况

| 场景                   | 处理                                                                       |
| ---------------------- | -------------------------------------------------------------------------- |
| 后端返回空数组         | `hasMore = false`，按钮禁用                                                |
| 后端返回数 < pageSize  | `hasMore = false`，按钮禁用                                                |
| 快速连续点击「上一页」 | 按钮 loading 时 disabled，防止并发                                         |
| 快速切换筛选条件       | `requestIdRef` 竞态保护，旧请求结果丢弃                                    |
| 图表数据为空           | ECharts 首次渲染前 formatter 收到 undefined → `v?.slice(5) ?? ''` 空值保护 |
| 窗口 resize            | `useEffect` 监听 resize → `chart.resize()`                                 |
| 弹框内翻页后关闭弹框   | 数据在 store 中，卡片自动同步                                              |
| 国债收益率首次加载     | 任一模块触发即写入缓存，其余模块命中                                       |

## 9. 迁移指南

1. 复制 `src/pages/dashboard/` 整个目录到目标项目
2. 安装依赖：`echarts` `dayjs` `zustand` `antd` `@ant-design/icons`
3. 注册路由：
   ```tsx
   { path: '/dashboard', element: <DashboardPage /> }
   ```
4. 替换真实 API：修改 `api.ts` 第 9-12 行 import 源
5. 调整 `constants.ts` 中 `DEFAULT_PAGE_SIZE` 匹配后端字典
6. 生产环境中 `constants.ts` 中的筛选选项应从后端字典接口动态获取
